软件源安装

ottohub-electron可以通过NPM软件源安装。

通过NPM软件源安装ottohub-electron，代码如下：

$ npm i ottohub-electron

开发环境
此软件要求Node版本大于等于22。
准备开发环境，代码如下：

$ npm i

调试代码如下：

$ npm start

打包代码如下：

$ npm run build

打包前要关闭占用8080端口的所有程序，否则将打包失败。

在Windows电脑中安装Github Desktop

网址：https://desktop.github.com/download/

在电脑中安装VSCode

网址：https://code.visualstudio.com/Download

在VSCode中安装Live Server插件

用于HTML实时调试

在VSCode中安装Vue (Official)插件

用于高亮Vue代码

在VSCode中安装Vue 3 Snippets插件

用于快速输入Vue代码

在VSCode中安装XML插件

用于高亮和补全所有HTML标签

在VSCode中安装文心快码插件

用于代码AI补全

正常的Git协作流程是：你fork这个Git仓库，在fork的仓库改完代码之后，在这个Git仓库提一个pull request。
此外，我可以给你这个项目的所有权限，这样你可以直接改这个项目的代码，免去fork和pull request，更加方便。所有人只要提一个issue，我就会给这个项目的所有权限。小朋友们可以在休闲的时候练一练编程，我给你们这个机会。
我的编程实际经验如下：

1. 要坚持。

2. 要提高联网搜索和抄代码的能力。不要重复造轮子。

3. 只要可以复制粘贴，就不要一个字母一个字母手打。

4. 多利用查找和批量替换功能。

5. 多测试（多做单元测试）。

6. 要精益求精。

7. 要劳逸结合。

8. 要注意坐姿。

9. 要注意清淡饮食。
